# CPU Scheduling Algorithms Simulator

An interactive desktop application built using **Python** and **CustomTkinter** to simulate and visualize various CPU scheduling algorithms. This tool is designed to help computer science and software engineering students understand how operating systems manage process scheduling in real-time.

---

## 🚀 Features

* **Interactive GUI:** Clean, modern, and dark-mode friendly user interface built with CustomTkinter.
* **Supported Algorithms:**
  * First-Come, First-Served (FCFS)
  * Shortest Job First (SJF) - Non-preemptive
  * Shortest Remaining Time First (SRTF) - Preemptive
  * Round Robin (RR)
* **Visual Gantt Chart:** Step-by-step graphical representation of how processes are allocated to the CPU.
* **Performance Metrics:** Automatically calculates and displays essential metrics:
  * Waiting Time for each process
  * Turnaround Time for each process
  * Average Waiting Time & Average Turnaround Time

---

## 🛠️ Tech Stack

* **Language:** Python
* **GUI Framework:** CustomTkinter (Extension of Tkinter for modern UI components)
* **Libraries:** Tkinter (Standard GUI Library)
